# 吾圈

吾圈树洞是一个移动端私密树洞 App，用 81 只萌宠虚拟好友陪你投递秘密、委屈、压力和没人方便听见的小心事。

## 电脑打开方式

双击 `index.html` 即可使用。

## 免费 CDN 部署方式

如果不想买 VPS，也不想备案，可以用 Cloudflare Pages + Pages Functions。说明见 `CLOUDFLARE.md`。

## 手机安装方式

这是一个 PWA 手机安装版。双击 `启动吾圈手机安装版.bat`，保持窗口不要关闭。

电脑本机可打开：

```text
http://localhost:5177
```

手机和电脑在同一个 Wi-Fi 时，用手机浏览器打开：

```text
http://电脑的IPv4地址:5177
```

然后安装到手机主屏幕：

- Android Chrome：菜单 -> 添加到主屏幕 / 安装应用
- iPhone Safari：分享 -> 添加到主屏幕

安装后就可以像普通手机 App 一样从主屏幕打开。

## 已包含功能

- 投递文字和本地图片树洞
- 本地持久化保存，刷新不丢
- 81 只真实生成萌宠头像
- 自动随机送抱抱
- 自动生成基于树洞内容的萌宠回声
- 配套 Node 后端代理 AI，请求密钥只保存在服务器环境变量里
- 搜索、按文字/图片/今天/收藏筛选
- 收藏树洞、删除树洞、重抽抱抱和回声
- 支持回复萌宠回声
- 支持和单只萌宠私聊
- 数据导出和导入 JSON 备份

## 文件说明

- `index.html`：App 页面
- `styles.css`：界面样式
- `app.js`：交互逻辑和本地存储
- `assets/avatars/`：81 张萌宠头像
- `assets/pet-avatars-spritesheet.png`：头像原始精灵图

## 隐私说明

默认情况下，所有记录都保存在浏览器本地 `localStorage` 中。只有投递树洞、重抽回声或萌宠私聊需要 AI 时，才会调用设置里的接口。
